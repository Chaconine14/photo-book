# Photo Book LP｜模写練習

## 概要
コーディング力向上を目的に、既存のLPを模写したポートフォリオです。  
FigmaでデザインカンプをトレースしながらHTML/CSS/Sassで実装しました。

---

## 制作の意図・学んだこと
- `margin` は要素間の余白、`padding` は要素内の余白と意識的に使い分ける練習をした
- Sassのディレクトリ構成（FLOCSS）を導入し、CSS設計を学んだ
- レスポンシブ対応を〇〇の方針で実装した（※実装していれば）
